# Technology responsibility matrix

| Technology | Responsibility |
|---|---|
| GSAP | Large narrative timelines and controlled transitions |
| GSAP Flip | Layout and shared-state transformations |
| ScrollTrigger | Pinned storytelling, scrub and progress |
| SplitText | Responsive editorial headline reveals |
| Observer | Wheel, touch, pointer and gesture navigation |
| Draggable + Inertia | Momentum-based workflow interaction |
| MorphSVG + MotionPath | 2D system diagrams and moving context |
| CustomEase | Consistent motion signature |
| ScrollToPlugin | Controlled navigation to product states |
| Lenis | Smooth scrolling synchronized with ScrollTrigger |
| Motion | Route-level reveal and small React transitions |
| Zustand | Shared product mode and drawer state |
| Radix UI | Accessible disclosure and primitive behavior |
| Vaul | Request-access drawer |
| React Hook Form + Zod | Validated conversion form |
| Sonner | User feedback |
| Embla | Accessible product proof carousel |

No 3D, WebGL or persistent canvas renderer is included.
