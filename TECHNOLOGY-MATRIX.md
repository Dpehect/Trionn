# Technology Matrix

- Next.js / React / TypeScript
- Tailwind CSS foundation plus custom editorial CSS
- GSAP core, Flip, ScrollTrigger, SplitText, Draggable, InertiaPlugin
- Motion for shared layout, spring drawer and answer transitions
- Raw WebGL fragment shader for the fabric field
- Lenis smooth scrolling
- Zustand persisted cart and wishlist
- Vaul cart drawer
- Sonner feedback
- Next/Image and local SVG product media

No Three.js, React Three Fiber, 3D models or permanent WebGL scene are used.
