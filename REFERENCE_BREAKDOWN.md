# Reference interaction breakdown

The supplied recording uses a fixed full-viewport editorial composition. A serif statement occupies the visual center, while equally cropped project thumbnails form asymmetrical groups around it. Hovering a thumbnail temporarily suppresses unrelated work, rearranges a selected project's media into a sparse collage and replaces the mission statement with the project name. Leaving the active area restores the original mosaic.

The rebuild uses CSS positioning and GSAP rather than Three.js. This keeps the effect faithful without a continuous WebGL render loop.
