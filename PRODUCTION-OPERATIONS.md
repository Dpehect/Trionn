# Production Operations

## Daily
- Review error logs
- Review inquiry delivery
- Check failed jobs

## Weekly
- Review Web Vitals
- Review authentication failures
- Verify storage growth
- Review audit logs

## Monthly
- Dependency updates
- Backup restore test
- RLS review
- Lighthouse regression
- Accessibility regression

## Incident response
1. Disable affected feature flag
2. Inspect readiness and version endpoints
3. Review structured logs and audit trail
4. Roll back Vercel deployment
5. Document root cause and prevention
