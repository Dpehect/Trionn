# Softbridge Design System

## Principles
1. Nordic restraint
2. Editorial scale
3. Motion with purpose
4. High contrast
5. Progressive enhancement

## Palette
- Polar: #F4F5EF
- Ink: #0C1412
- Baltic: #184B63
- Aurora: #A6FF8F
- Mist: #CBD7D1

## Motion
Fast 180ms, regular 420ms, cinematic 900ms. Primary easing: cubic-bezier(.22,1,.36,1).
