# Accessibility Standard

Target: WCAG 2.2 AA.

## Implemented
- Skip link
- Visible focus states
- Route announcements
- Reduced motion support
- Keyboard menu handling
- Focus trap primitive
- Carousel control primitive
- Semantic page structure
- Canvas fallback

## QA checklist
- Keyboard-only navigation
- Screen reader page titles
- Form error associations
- 200% zoom
- High contrast mode
- Reduced motion
