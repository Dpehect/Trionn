# Phases 01–04 Delivery

## Phase 01 — Strategy
Finland-first positioning, Nordic clarity, English-first communication, Helsinki/Espoo/Tampere/Turku market framing and conversion architecture.

## Phase 02 — Architecture
Server-first Next.js App Router structure with client islands only for animation and interaction. Content is separated from presentation.

## Phase 03 — Design System
Aurora-inspired accent, polar neutrals, strict typographic rhythm, 12-column layout, accessible contrast and motion tokens.

## Phase 04 — Global Experience
Floating navigation, full-screen menu, responsive shell, smooth scroll, cursor treatment, page progress, footer and reduced-motion handling.

## Acceptance criteria
- `npm run build` succeeds
- Keyboard reachable navigation
- Mobile menu traps visual context and closes on navigation
- Motion disabled when users prefer reduced motion
- No copied Zentry branding or proprietary assets
