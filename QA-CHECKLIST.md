# QA checklist

- 320, 375, 768, 1024, 1440 and 1920 pixel layouts
- Keyboard navigation and visible focus states
- `prefers-reduced-motion` fallback
- Touch, wheel and pointer Observer behavior
- Draggable bounds after viewport resize
- GSAP context and plugin cleanup
- Drawer focus management and form validation
- Route changes and browser back/forward
- Safari backdrop-filter and sticky behavior
- CLS, image-free first render and no persistent render loop
- TypeScript, ESLint and production build
