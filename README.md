# Softbridge Solutions — Finland Agency Website

Production-oriented foundation covering phases 1–4: strategy, architecture, design system, navigation and global experience.

## Run
```bash
npm install
npm run dev
```

## Included
- Next.js 16 App Router + React 19 + TypeScript
- Tailwind CSS v4 design tokens
- GSAP + ScrollTrigger infrastructure
- Lenis smooth scrolling
- Responsive floating navigation and full-screen menu
- Nordic visual system and reusable UI primitives
- SEO metadata, sitemap, robots, loading and 404 states

See `docs/PHASES-01-04.md` and `docs/DESIGN-SYSTEM.md`.
