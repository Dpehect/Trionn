# Studio Freight Reference Rebuild — 14 Complete Phases

This package contains fourteen standalone Next.js projects. Every phase includes its own package configuration, source code, local WebP media, responsive styling and documentation.

The final experience reproduces the supplied video's main interaction language:

- fixed editorial navigation and footer
- central serif mission statement
- twenty-image project mosaic
- project hover grouping
- GSAP / GSAP Flip focus transitions
- SplitType title reveals
- custom cursor and micro-interactions
- project detail pages and studio content pages
- mobile and reduced-motion fallbacks

Start with any phase folder:

```bash
npm install
npm run dev
```

For the completed version, open `phase-14-final-reference-polish`.
