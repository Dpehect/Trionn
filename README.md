# SABLE Fashion Commerce — Phase 14

    A professional editorial fashion storefront built with Next.js, React, TypeScript, Tailwind CSS, GSAP, Lenis, Motion, Zustand, React Hook Form, Zod, Radix UI, Vaul, Sonner and Embla-ready architecture.

    ## Included scope

    - Phase 01: Foundation & Design System
- Phase 02: Editorial Homepage
- Phase 03: GSAP Motion System
- Phase 04: Shop & Collections
- Phase 05: Product Detail Experience
- Phase 06: Cart, Wishlist & Checkout
- Phase 07: Lookbook, Campaigns & Journal
- Phase 08: Customer Account
- Phase 09: Admin Commerce Console
- Phase 10: Content Studio & CMS Layer
- Phase 11: Premium UX & Accessibility
- Phase 12: Performance & SEO
- Phase 13: Responsive Polish & QA
- Phase 14: Production Release

    ## Run

    ```bash
    npm install
    npm run dev
    ```

    Production validation:

    ```bash
    npm run typecheck
    npm run lint
    npm run build
    ```

    ## Project notes

    - This ZIP is a complete snapshot and does not depend on previous phases.
    - Product and editorial visuals are local SVG assets, so the demo runs without image hosting.
    - Commerce data currently comes from typed local data and a replaceable service layer.
    - Three.js is intentionally excluded. The cinematic experience uses GSAP, ScrollTrigger, SplitType, Lenis, Motion and image choreography instead of a persistent WebGL canvas.
    - Replace the placeholder brand, copy and generated campaign studies before production launch.
